# Portfolio Website

## Overview

A high-performance personal portfolio website built with Next.js 15, Tailwind CSS, and TypeScript.
Features a premium dark-mode design with advanced animations using Framer Motion.

## Technologies

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **Zod** (Validation)

## Features

- **Hero Section**: Anti-gravity float animation using Framer Motion.
- **Tech Stack**: Interactive grid with hover effects.
- **Projects**: 3D hover-tilt cards.
- **Contact**: Functional form with Zod validation and visual feedback.

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure

- `src/app`: Page layouts and global styles.
- `src/components`: Reusable UI components (Hero, TechStack, Projects, Contact).
- `public`: Static assets (images).
